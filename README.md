# Online Examination Systen  
## Setup  
1. Run `npm install`  
2. Install knex globally: `npm install -g knex`  
3. Create knexfile: `knex init` and configure it
4. Run migrations: `knex migrate:latest`

## Production launch
1. Compile vue: `npm run build`
2. Run server: `npm start`

## Development launch
1. `npm run serve`
